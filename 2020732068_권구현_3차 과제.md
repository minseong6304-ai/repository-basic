3x3 행렬의 상태천이행렬은 손으로 구하기 어렵기 때문에, 매트랩으로 진행  
```  
syms s
A = [1,1,-1;4,3,0;-2,1,10];
B = [0;0;4];
C = [1,0,0];
D = 0;

phi = inv(s*eye(3) - A);
G = C*phi*B+D;
[n, d] = numden(G)

% 검산
num = sym2poly(n)
den = sym2poly(d)
[A, B, C, D] = tf2ss(num, den)  
```
